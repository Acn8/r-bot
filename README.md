# 支持环境
## Linux/amd64,Linux/arm64,Linux/ppc64le,Linux/s390x,Linux/arm/v7

# 开始部署
部署并启动容器
```
docker run -d --name bot -p 9527:9527 wuge6/bot
```
进入容器  
```
docker exec -it bot /bin/bash
```
编辑`client_config`配置文件  
```
vi client_config
```
选择要上传的`pem`密钥  
`rz`  
重启机器人  
```
bash sh_client_bot.sh
```
## 退出容器
`exit`  

# 删除容器和镜像
```
docker rm -f bot
```
```
docker rmi -f wuge6/bot:v1
```
